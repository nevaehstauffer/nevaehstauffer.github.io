Does Nebraska Men’s Baskebtall Shoot Better Behind the 3 Point Line on their Home Court?

nebraska
basketball
shooting
Author

Nevaeh Stauffer

Published

November 10, 2023

For the last five seasons, only 1 of 5 have ended with a winning record and that was back in the 2018-2019 season. In the next four, the best record they have had was when they went 16-16 last season. It seems Nebraska’s men’s basketball team shoots way better when they play at home compared to when they’re away. Is it their home atmosphere or just the familiarity of PBA? Let’s see if numbers can back up this idea.

Sports Reference is a sports statistics website that provides the shooting percentages over each of the last five seasons. You can see the percentages of overall shots, free throws, and three-pointers. Within the data, it is easy to calculate Nebraska and the rest of college basketball’s shooting averages while differentiating the home and away games. To determine whether there is a clear answer about where the Husker’s shoot best, let’s look at Nebraska’s home and away shooting percentages from the last 5 seasons.

Code 
library(tidyverse)
library(ggrepel)
library(ggalt)
library(ggbeeswarm)
library(gt)

men2019 <-read_csv("neb.2019.csv")
men2020 <-read_csv("neb.2020.csv")
men2021 <-read_csv("neb.2021.csv")
men2022 <-read_csv("neb.2022.csv")
men2023 <-read_csv("neb.2023.csv")

men <- bind_rows(men2019,men2020,men2021, men2022, men2023)

menthrees <- men |> 
  group_by(Location, Year) |> 
  summarize(
    Average3PPct = mean(ThreePpct)
  )
mentwos <- men |> 
  group_by(Location, Year) |> 
  summarize(
    AverageFGpct = mean(FGpct)
  )

ggplot() + 
  geom_line(data = menthrees, aes(x = Year, y = Average3PPct, color = Location, group=Location)) +
  labs(
    title = "Nebraska Shoots better with the home court advantage",
    subtitle = "1 out of the 5 last seasons their 3P% on the road is worse",
    x = "Year",
    y = "Averages",
    caption="Source: Sports Reference | Nevaeh Stauffer"
    ) +
  theme_minimal()+
    theme(
    plot.title = element_text(color = "red", size = 14, face = "bold"),
    plot.subtitle = element_text(color = "black", size = 12, face = "bold")
  )

Nebraska has shot significantly better on their home court behind the three point line for 4 out of the 5 past seasons. During the 2021-2022 season the away percentage was better but not by much.

Now let’s compare the same data about Nebraska’s last five seasons to the rest of the teams in the Big 10 and see where they stand.

Code 
logs23 <- read_csv("https://mattwaite.github.io/sportsdatafiles/logs23.csv")
logs22 <- read_csv("https://mattwaite.github.io/sportsdatafiles/logs22.csv")
logs21 <- read_csv("https://mattwaite.github.io/sportsdatafiles/logs21.csv") 
logs20 <- read_csv("https://mattwaite.github.io/sportsdatafiles/logs20.csv") |> 
  rename(Season = season, TeamFull=Team)
logs19 <- read_csv("https://mattwaite.github.io/sportsdatafiles/logs19.csv") |> 
  rename(Season = season, TeamFull=Team)

logs <- bind_rows(logs23, logs22, logs21, logs20, logs19) |> 
  mutate(
    Location = case_when(
      is.na(HomeAway) == TRUE ~ "Home",
      HomeAway == "@" ~ "Away",
      HomeAway == "N" ~ "Neutral"
    )
  )

homeawaylogs <- logs |> 
  group_by(TeamFull, Conference, Season, Location) |>
  summarize(
    Average3PPct = mean(Team3PPCT, na.rm=TRUE)
  ) |> 
  mutate(TeamLocation = paste(TeamFull, Location))

big10away <- homeawaylogs |>
  filter(str_trim(Conference) == "Big Ten") |> 
  filter(Location == "Away")


big10home <- homeawaylogs |>
  filter(str_trim(Conference) == "Big Ten") |> 
  filter(Location == "Home")

nuhome <- big10home |> 
  filter(TeamFull == "Nebraska Cornhuskers")
nuaway <- big10away |> 
  filter(TeamFull == "Nebraska Cornhuskers")
  
menaway <- menthrees |> 
  group_by(Location, Year) |> 
  filter(Location == "Away")

menhome <- menthrees |> 
  group_by(Location, Year) |> 
  filter(Location == "Home") 

ggplot() +
  geom_beeswarm(
    data=big10home, 
    aes(x=Average3PPct, y=Season), color="darkblue", alpha=.3) +
  geom_beeswarm(
    data=big10away, 
    aes(x=Average3PPct, y=Season), color="blue", alpha=.3) +
  geom_beeswarm(
    data=nuhome, 
    aes(x=Average3PPct, y=Season), color="red") + 
  geom_beeswarm(
    data=nuaway, 
    aes(x=Average3PPct, y=Season), color="darkred") +
  geom_text_repel(data=nuhome, 
    aes(x=Average3PPct, y=Season, label=TeamLocation)) + 
  labs(
    title = "Nebraska's Three Point shooting is just average ",
    subtitle = "Comparing the Huskers to the Big Ten, Nebraska is in the middle",
    x = "Averages",
    y = "Season",
    caption="Source: Sports Reference | Nevaeh Stauffer"
    ) +
  theme_minimal()+
    theme(
    plot.title = element_text(color = "red", size = 14, face = "bold"),
    plot.subtitle = element_text(color = "black", size = 12, face = "bold")
  )

So we know that Nebraska shoots better behind the arc with the home court advantage. But compared to the rest of the Big Ten, those percentages still aren’t great and are just about average. Now let’s see if Nebraska’s best home shooting percentages rank in the top 10 in the Big Ten over the past five seasons.

Code 
big10home |> 
  ungroup() |> 
  top_n(10, wt=Average3PPct) |> 
  arrange(desc(Average3PPct)) |>
  select(TeamFull, Season, Average3PPct) |> 
  gt() |> 
  cols_label(
    TeamFull = "Team",
    Average3PPct = "Percentage"
  ) |>
  tab_header(
    title = "Behind the arc isn't where the Huskers are most likely to dominate",
    subtitle = "Nebraska's best 3 Point Percentage doesn't even rank in the Top 10"
  ) |> tab_style(
    style = cell_text(color = "black", weight = "bold", align = "left"),
    locations = cells_title("title")
  ) |> tab_style(
    style = cell_text(color = "black", align = "left"),
    locations = cells_title("subtitle")
  ) |>
  tab_source_note(
    source_note = md("**By:** Nevaeh Stauffer  |  **Source:** Sports Reference")
  ) |>
  tab_style(
     locations = cells_column_labels(columns = everything()),
     style = list(
       cell_borders(sides = "bottom", weight = px(3)),
       cell_text(weight = "bold", size=12)
     )
   ) |>
  opt_row_striping() |> 
  opt_table_lines("none") |>

    columns = Average3PPct,

  tab_style(
    style = list(
      cell_fill(color = "red"),

  )

So Nebraska has had some great individual shooters on their team but as a team, behind the three point line isn’t where you are gonna see them perform at their best abilities.
